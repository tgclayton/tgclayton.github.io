Content

All elements in css are made of boxes. Each of these boxes is composed of several differant layers wrapped around each other, each of which can be seperately styled and sized.

This box visualises an element and its different layers, the blue box in the centre is the content. This box contains the actual content of the element, such as text or images. The size of the content box is controlled with the <height> and <width> tags, while its colour is controlled with the <background-color> tag.

Padding

This box shows the padding in green. The padding is the space immediately around the content, often used to put distatance between the content and the borders of the element or to center content inside a larger element.

The padding box cannot normally be coloured seperately from the content but I have shown it here using an internal box-shadow with the background colour attached only to the content box with background-clip;.

Borders

This box shows the border component highlighted in red. The border is the area outside of the padding and before the margin begins. The border can be any colour and multiple different styles such as dashed or solid or doubled.

The properties of the can be set with various tags such as border-color and border-style or by combinin all properties under the single border tag as I have done here. The border show is defined by the code "border: 2px solid red;"

Margins

This box shows the margin in yellow. The margin is the area on the outside of the element that is usually used for position and for seperating elements from each other, this works because the margin pushes the element away from other elements.

The margin is invisble and cannot be styled, I have highlighted it here by colouring the background of the containing element.
